
# Test Technique Seomaniak - Dashboard Utilisateurs (React)

Ce projet est une petite application React qui permet de gérer des utilisateurs à travers un dashboard simple.

## Fonctionnalités

- Ajouter un utilisateur
- Modifier un utilisateur
- Supprimer un utilisateur
- Afficher la liste des utilisateurs
- Afficher des statistiques (Total, Actifs, Inactifs)
- Sauvegarde des données avec localStorage

## Technologies utilisées

- React
- Vite
- CSS
- localStorage

## Lancer le projet

1. Installer les dépendances

npm install

2. Lancer le serveur de développement

npm run dev
